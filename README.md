# HTML Tables Lab

## Overview

In this lab, you will practice creating HTML tables to display structured information.

Imagine that you have been asked to add a drinks menu to the **LearnThenCode Café** website. Rather than displaying the menu as plain text, you will organize it into a professional HTML table.

This lab focuses on building well-structured tables using the appropriate HTML elements.

---

## Learning Objectives

By completing this lab, you should be able to:

- Create an HTML table.
- Add table headers.
- Organize tables using `<thead>` and `<tbody>`.
- Add a descriptive table caption.
- Display structured data using rows and cells.

---

## Your Task

Inside the `starter/index.html` file, create a table that represents the **LearnThenCode Café Drinks Menu**.

Your table should satisfy all of the requirements below.

### Requirements

Your table must include:

- A `<table>` element.
- A `<caption>` describing the table.
- A `<thead>` section.
- A `<tbody>` section.
- A header row containing:
  - **Drink**
  - **Price**
- At least **three** drinks.
- A price for each drink.

---

## Example Menu

| Drink         | Price   |
|---------------|---------|
| Tea           | KES 150 |
| Coffee        | KES 200 |
| Hot Chocolate | KES 250 |

You may use the values above or create your own menu.
